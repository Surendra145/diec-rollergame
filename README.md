# 🎲 Dice Roller Game

A simple Python project to roll two dice randomly.

---
## ✅ Conditions
- Type `y` ➔ Roll the dice.
- Type `n` ➔ Exit the game.
- Any other input ➔ Shows **"Invalid choice!"** message.

---## 🛠️ Tech Stack
- Python 3
- random module (built-in)

Enjoy the game! 🎲
